### データのあれこれ
test2.pyでスクレイピングしたデータから累積データを取得.算出するデータは以下の通り.
- HR(本塁打数)
- R(得点)
- RBI(打点)
- SB(盗塁)
- H(安打)
- BB/HBP(四死球)
- SO(三振)
- PA/AB(打席/段数)
- AVG(打率)
- OBP(出塁率)
- SLG(長打率) 
- ISO(長打力)
- BABIP(インプレー打球の安打率)
